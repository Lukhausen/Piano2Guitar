Muted Strings Permutation
When this setting is enabled, the chord generator will consider every possible configuration of muted strings, even if some strings could be played.
If disabled, the chord generator avoids unnecessarily muting strings.
Start with Root
If enabled, the lowest note of a chord will always be the root note.
If disabled, chords can start with any note that is part of the chord.
